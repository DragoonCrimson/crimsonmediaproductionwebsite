What is C++

C++ is an all-purpose coding language commonly used in app development, it is across platform and very easy to manipulate.
It was created by Bjarne Stroustrup in 1980 as a High-level programming language. It is one of the most common Object Oriented Programming languages, also known as OOP. Others along this line include:

    Hello World

    This example code is used to print the basic "Hello World", as a standard ruling since the WorldWideWeb became available Hello World has been used to state a test of whether a string outputs.

    Using a standard format for a C++ file
    using namespace std;
    cout << "Hello World << endl;
    This prints a standard hello world as a string

    Taking User Input

    This set of code will use a variable to take in a form of user input and output it into a string format

    using namespace std;
    String userInput = "";
    cout << "Enter your Input in the next line" << endl;
    cin >> userInput;
    cout << "Your input is " + userInput << endl;

    Variables

    Some people may ask, but what is a variable, this code will use one of each MAIN variables, do not forget there are variations (will be noted underneath as a footnote)
    using namespace std;
    int numberOne = 1;
    string hello = "Hello";
    char Alpha = 'A';
    bool response = true;
    double decimal = 1.2;
    float accurateDecimal = 1.222;
    cout << numberOne << endl;
    cout << hello << endl;
    cout << Alpha << endl;
    cout << response << endl;
    cout << decimal << endl;
    cout << accurateDecimal << endl;
    The above lines of code will print out in order: 1, Hello, A, true, 1.2, 1.222
