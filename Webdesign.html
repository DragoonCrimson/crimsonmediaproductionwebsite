Welcome to HTML/CSS

What is HTML?

Some of you may be asking what is HTML and what is it used for.
Well to put it simply, HTML is a coding language used in the desigining of a website, it is not however the only language, but is the most common.

To make things more complicated, HTML stands for HYPERTEXT MEDIA LANGUAGE, it is a format of code aimed at designating certain commands to the computer and how the internet processes the information.

What is CSS?

CSS to put it simply is the face of the code used in website design and creation. This website that you are currently on is a good example of how HTML and CSS combine.

This page in of itself is coded in HTML and is using CSS to explain how the layout works.

But what is CSS? CSS stands for Cascading Style Sheet, it is a set of code aimed at defining the set parameters of each aspect of the webpage, whether a simple colour change, a layout change or an interactive element such as the navigation bar above.
